2025华为杯通信算法大赛

AI使能的无线鲁棒SVD算子 | 西北赛区一等奖（前1%）
  1. 问题背景：MIMO信道矩阵SVD分解实时性需求，设计轻量化神经网络替代传统数值算法。
  2. 核心创新：受SVD左右奇异向量分别张成列空间与行空间启发，采用双路Transformer编解码架构——行路径建模行依赖生成U，列路径提取列特征生成V，结构与SVD子空间分解数学对齐。U/V解码器结合线性层与可微正交引导模块加速正交性收敛，S解码器融合双路隐状态预测相对奇异值并以Frobenius范数动态缩放实现信道自适应。
  3. 训练策略：复合损失联合优化重建误差与正交性，正交损失权重从0.06平滑增至2.0避免初期梯度冲突，配合鲁棒归一化提升收敛稳定性。参数优化后在保持高精度同时降低模型复杂度，重建误差降低53.6%。

科研项目

AeroRay

LLM驱动的低空电磁仿真智能孪生平台 | 平台演示视频
  1. 核心创新：首个将大语言模型与低空电磁仿真深度融合的全栈平台。通过LangGraph ReAct Agent将WinProp射线追踪引擎封装为自然语言交互接口，用户通过对话完成从地理数据提取、建筑场景建模、多基站三维射线仿真到无人机航线规划的全链路操作。
  2. 核心功能：低空多基站三维仿真、干扰态势4模式分析（RSRP/主导基站/SINR/重叠覆盖）、多基站感知航线规划、蜂群协同规划（1-10架无人机）。
  3. 技术架构：React 18 + Three.js前端3D渲染，FastAPI + LangGraph后端，C++ WinProp SDK仿真引擎，双通道WebSocket实时通信。
React Three.js FastAPI LangGraph C++ WinProp SDK WebSocket Claude

RadioDiff-Flux

基于扩散模型中间点重用的高效射频地图构建
  1. 核心发现：相似场景的扩散过程中间点（midpoint）高度一致，中间状态主要编码静态环境特征（建筑结构、材料属性），后期阶段负责动态细化（天线辐射模式、动态障碍物）。
  2. 方法创新：提出两阶段隐式扩散框架——第一阶段仅用静态场景特征生成粗粒度隐表示并缓存共享，第二阶段用预训练模型将其适配到动态条件和发射机位置，避免重复早期计算。
  3. 性能突破：实现50倍推理加速，精度损失<0.15%，显著降低动态6G场景的延迟，保持生成保真度。基于KL散度理论分析证明相似结构射频地图的去噪轨迹紧密对齐。
PyTorch Diffusion Model Latent Diffusion Midpoint Reuse 6G Networks

RadioMamba

打破射频地图构建精度-效率权衡的混合Mamba-UNet架构
  1. 问题分析：准确射频地图构建需建模长程空间依赖（反映波传播物理全局性），CNN局部感受野不足，Transformer全局注意力复杂度二次方。
  2. 架构创新：提出RadioMamba混合Mamba-UNet，核心为Mamba-Convolutional块——Mamba分支以线性复杂度捕获全局依赖，并行卷积分支提取局部特征，生成兼具全局上下文和局部细节的表示。
  3. 性能突破：相比RadioDiff等扩散模型精度更高（所有指标SOTA），速度快近20倍，参数量仅2.9%，支持实时6G系统部署。验证了Mamba架构在物理信息任务中的独特适配性。
PyTorch Mamba State Space Models UNet Linear Complexity

RadioMapMotion

首个时空射频地图预测数据集与基准
  1. 数据集创新：首个大规模连续射频地图序列数据集，包含物理一致的车辆轨迹驱动的信号演化（非独立静态快照），保留车辆运动与信号变化的因果关系，为数据驱动预测建模提供基础。
  2. 任务定义：从历史观测预测未来射频地图序列，从"构建当前状态"推进到"预测未来演化"，支持前瞻性网络操作。
  3. 基准模型：提出RadioMotionNet（ConvLSTM-UNet混合架构），利用历史测量联合保持空间保真度和时序连续性，无需实时环境感知。实验证明高预测精度、结构保真度和低推理延迟，适合实时前瞻性网络管理。
PyTorch ConvLSTM Spatio-Temporal Dataset Proactive Network

UrbanMIMOMap

首个大规模城市MIMO CSI数据集
  1. 数据集特色：基于高精度射线追踪生成，提供完整复数4×4 MIMO信道矩阵（H），0.5m空间分辨率，真实城市几何（OpenStreetMap），支持预编码感知RSS和理论信道容量等性能指标推导。
  2. 核心优势：原生MIMO配置（非SISO合成），完整复数矩阵（含精确幅度相位），亚米级分辨率，真实大规模城市场景，预渲染可视化地图支持图像深度学习，灵活参数配置框架。
  3. 应用价值：为数据驱动射频地图生成、MIMO空间性能分析、6G环境感知研究提供关键数据集和参考基准（含RadioUNet等代表性ML方法基线性能）。
Ray Tracing MIMO CSI Dataset WinProp OpenStreetMap

Semantic Communication Framework

面向6G的语义通信框架
  1. 研究背景：研究面向6G的语义通信理论与实现。提出基于深度学习的端到端语义编解码框架，在发送端提取信息语义特征（而非比特流），通过信道传输语义表示，接收端根据语义重建原始信息。
  2. 系统设计：针对图像传输场景，设计语义编码器（ResNet + Attention）提取关键语义（物体、场景、关系），信道编码器适配无线信道特性，接收端语义解码器重建图像。
  3. 性能提升：相比传统通信，在低信噪比下传输效率提升3-5倍，同时保持语义保真度。
PyTorch Semantic Communication Autoencoder Channel Coding 6G

Multi-Agent Air Combat Simulation

基于NetLogo的多智能体空战模拟
  1. 系统构建：基于NetLogo平台构建多智能体空战仿真系统，研究无人机集群协同作战策略。实现红蓝双方多架无人机的自主决策、态势感知、协同攻击与防御机制。
  2. 决策架构：智能体采用分层决策架构——战术层基于强化学习（DQN）学习攻击/规避策略，协同层通过通信网络共享目标信息实现集群协作。
  3. 仿真验证：仿真环境支持3D空域、导弹追踪、电子对抗等要素。通过大量对抗实验验证，协同策略相比独立作战胜率提升40%+。
NetLogo Multi-Agent System Reinforcement Learning Swarm Intelligence
